Student Health & Performance Predictor

This project is a machine learning based system that predicts student academic performance and health risk based on daily lifestyle habits.

Description
The system takes inputs such as study hours,sleep hours,exercise and screen time and uses machine learning models to:
-Predict student marks (Regression)
-Classify health risk (High/Low)(Classification)
-Show performance trend using a graph (Forecasting)

Technologies:
-Python  
-Pandas  
-Scikit-learn  
-Matplotlib  
-Streamlit  

How to Run
1.Install libraries:
   pip install pandas scikit-learn matplotlib streamlit
   
2.Run CLI version:
   python model.py

3.Run UI version:
   streamlit run app.py

Features:
-Predicts marks based on lifestyle habits  
-Detects health risk  
-Displays performance graph
